# About App
This App Using API https://www.thecocktaildb.com/api.php

## Feature
- Showing Cocktail
- Displays a list of cocktail based on the selected category
- Displaying cocktail Details based on the selected item cocktail
- Searching cocktail
- Favourite cocktail

## Tech
- Kotlin
